Format Kolom Excel
Pastikan file memiliki kolom-kolom berikut:
barcode nama satuan stok harga_beli harga_ecer harga_grosir min_beli_grosir keterangan
Kolom keterangan bersifat opsional. Jika barcode sudah ada di gudang, data akan diperbarui. Jika belum, barang akan ditambahkan sebagai data baru.
Langkah Upload
1
Download Template Gunakan template resmi agar format kolom sesuai.
2
Isi Data di Excel Lengkapi barcode, nama barang, harga, dan stok. Simpan sebagai .xlsx.
3
Upload & Preview Pilih file lalu klik Upload. Sistem akan menampilkan preview data sebelum disimpan.
4
Konfirmasi Simpan Cek preview, lalu klik Konfirmasi untuk menyimpan ke Gudang Utama.

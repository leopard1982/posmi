Pilih File Template:

Catatan:

  1. Silakan download file template dan mengisikan data barang
  2. Apabila ada barang dengan kode barcode yang sama akan dilakukan update harga dan jumlah stok, jadi stok lama akan berubah
  3. Setelah Upload untuk melakukan pengecekan barang silakan masuk ke menu Barang → Daftar Barang
  4. File yang diperkenankan adalah tipe xlsx atau xls (file excel)
